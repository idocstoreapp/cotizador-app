# Instrucciones para Imágenes de la Landing Page

## Estructura de Carpetas

Crea la siguiente estructura en la carpeta `public`:

```
public/
  imagenes/
    cocina-hero.jpg          (Imagen principal del hero - fondo)
    cocina.jpg                (Imagen de área: Cocina)
    closet.jpg                (Imagen de área: Closet)
    recibidor.jpg             (Imagen de área: Recibidor)
    sala.jpg                  (Imagen de área: Sala)
    remodelacion-locales/     (Carpeta con imágenes de remodelaciones)
      imagen1.jpg
      imagen2.jpg
      ...
    logos/                    (Carpeta con logos de clientes/partners)
      logo1.png
      logo2.png
      ...
```

## Imágenes Requeridas

### Hero Section
- **cocina-hero.jpg**: Imagen de fondo para el hero (recomendado: 1920x1080px)
  - Debe ser una imagen de cocina moderna y atractiva
  - Se usará con opacidad reducida como fondo

### Carrusel de Áreas (4 imágenes)
- **cocina.jpg**: Imagen representativa de cocina (recomendado: 400x300px)
- **closet.jpg**: Imagen representativa de closet (recomendado: 400x300px)
- **recibidor.jpg**: Imagen representativa de recibidor (recomendado: 400x300px)
- **sala.jpg**: Imagen representativa de sala (recomendado: 400x300px)

### Logos de Confianza
- Coloca los logos en formato PNG con fondo transparente
- Tamaño recomendado: 200x80px aproximadamente
- Nombres sugeridos: logo1.png, logo2.png, logo3.png, etc.

## Notas

- Todas las imágenes deben estar optimizadas para web
- Formato recomendado: JPG para fotos, PNG para logos
- Las imágenes se cargarán automáticamente cuando las coloques en la carpeta `public/imagenes/`
- Si no existen las imágenes, se mostrarán espacios en blanco o placeholders

## Actualización

Una vez que agregues las imágenes reales, la landing page las mostrará automáticamente sin necesidad de modificar el código.
